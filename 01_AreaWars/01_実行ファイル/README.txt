|||||◆ 操作方法 ◆|||||||||||||||||||||
========================================
◇ゲームパッド(X-INPUT)

=====ゲーム画面=====
・移動         : 左スティック
・ショット     : 右スティック (※倒し続けることで連射可能)
・ボム         : RT or RB ボタン
・ポーズ       : START ボタン
=====メニュー画面===
・選択         : 左スティック or 十字ボタン
・決定         : A ボタン
====================

◇キーボード

=====ゲーム画面=====
・移動         : WASD キー
・ショット     : 左クリック (※長押しすることで連射可能)
・ボム         : 右クリック
・ポーズ       : P キー
=====メニュー画面===
・選択         : WS or 十字 キー
・決定         : ENTER キー
=====その他=========
・全画面切替   : F11 キー
・終了         : ESC キー
====================

========================================
|||||◆ 攻略方法 ◆|||||||||||||||||||||
========================================
◇スコアを稼ぐコツ
・金色の敵は高スコアです。積極的に狙いましょう。
・敵の爆発が連鎖することで、スコアの倍率が伸びていきます。
  敵が並んだタイミングを活用しましょう。
・ボムの最大所持数に達している時、敵が落とすボムはスコアアイテムに変化します。
・ゲームクリア時には「ライフボーナス」が発生します。
  被弾を最小限に抑えることでより多くのスコアを獲得できます。
