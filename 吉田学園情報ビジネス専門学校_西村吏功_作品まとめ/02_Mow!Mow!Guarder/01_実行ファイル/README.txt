|||||◆ 操作方法 ◆|||||||||||||||||||||
========================================
◇ゲームパッド(X-INPUT)

=====ゲーム画面=====
・移動         : 左スティック
・視点操作     : 右スティック
・アイテム切替 : LB / RB ボタン
・アイテム使用 : RT ボタン
・調べる/決定  : A ボタン
・ポーズ       : START ボタン
=====メニュー画面===
・選択         : 左スティック or 十字ボタン
・決定         : A ボタン
====================

◇キーボード

=====ゲーム画面=====
・移動         : WASD キー
・視点操作     : 十字キー	or マウス
・アイテム切替 : Q / E キー
・アイテム使用 : SPACE キー
・調べる/決定  : ENTER キー	or マウス左ボタン
・ポーズ       : P キー
=====メニュー画面===
・選択         : WS or 十字 キー
・決定         : ENTER キー
=====その他=========
・全画面切替   : F11 キー
・終了         : ESC キー
====================
※PCの性能によっては、同時に複数のキーを押しても反応しないことがあります。
  このような場合は、マウスを使って視点を操作することをお勧めします。
  マウスを使うと、左のボタンがENTERキーの代わりになります。

========================================
|||||◆ 攻略方法 ◆|||||||||||||||||||||
========================================
◇基本の動き
・ゲームの流れは「インターバル」と「襲撃」が交互に繰り返されます。
・インターバルでは、商品台からユニットや強化アイテムを購入して、次の襲撃に備えます。
・襲撃では、ウシさんがダメージを受けないように戦います。
  敵はプレイヤーかウシさんのどちらかを狙うため、
  周囲の強化が不十分の際はプレイヤーとウシさんの射程が一致しないように立ち回ることが望ましいです。
  また、あえて射程内で待ち構えて剣で弾を斬ることも有効な戦法です。

◇スコアを稼ぐコツ
・各ウェーブの終了時には「タイムボーナス」が発生します。より早くウェーブを終えるほど多くのスコアが得られます。
