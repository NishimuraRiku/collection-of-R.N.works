========================================
|||||◆ 操作方法 ◆|||||||||||||||||||||
========================================
◇キーボード

=====モデル操作=====
・モデルの種類変更       : [1][2] キー
・モーション変更         : [3][4] キー
・モデルデータ再読込     : [F3] キー
・別のモデルデータ読込   : [F4] キー
=====カメラ操作=====
・カメラの上下降         : [U][J] キー
・カメラの視点操作       : 十字キー or 右ドラッグ
=====その他=========
・テキスト   表示/非表示 : [F1] キー
・当たり判定 表示/非表示 : [F2] キー
・ポーズ                 : [P] キー
・終了                   : [ESC] キー
====================

========================================
|||||◆ 使い方 ◆|||||||||||||||||||||||
========================================
・実行すると、サンプルモデルのデータが開かれます。
・新しいデータを作成する場合は、F4キーでモデルデータを読み込み、
  MODELDATA内のTEMPLATE.txtをコピーして新しいプロジェクトを作成してください。
・モデルデータを読み込むと、自動的に内部テキストが開かれます。
  テキストに直接データを書き込んでいき、F3キーで再読み込みすることで変更が反映されます。

========================================
|||||◆ テキスト記述方法解説 ◆|||||||||
========================================
#(※"#"はコメントです。)

TYPE #00(※TYPE~TYPE_ENDまでの括りを増やすことで種類を追加出来ます。)
	
	PARTSSET #部品設定情報
		
		PARTS #00部品1(※PARTS~PARTS_ENDまでの括りを増やすことで部品を追加出来ます。)
			MODEL_PATH:			data\MODEL\TEST\head_x0.1.x     #モデルのパス
			PARENT:				-1                              #親パーツの番号(-1で始点)
			RELATIVE_POS:		0.0 0.0 0.0                     #親パーツに対する相対位置
			RELATIVE_ROT:		0.0 0.0 0.0                     #親パーツに対する相対向き(1につき半周)
			ROT_RELATIVE_ROT:	0.0 0.0 0,0                     #回転の視点位置(※使いにくいので触らないのが無難)
		PARTS_END
		
	PARTSSET_END
	
	HITTESTSET #当たり判定設定情報
		
		HITTEST #00==========通常==========
			
			HITTEST_PARTS #00部品1(※HITTEST_PARTS~HITTEST_PARTS_ENDまでの括りを増やすことで当たり判定を追加出来ます。)
				FORM:			1               #当たり判定の形(1:矩形)
				PARENT:			0               #親パーツの番号(-1で始点)
				WIDTH:			8.0             #幅
				HEIGHT:			8.0             #高さ
				DEPTH:			8.0             #奥行き
				RELATIVE_POS:	0.0 4.0 0.0     #親パーツに対する相対位置
				RELATIVE_ROT:	0.0 0.0 0.0     #親パーツに対する相対向き(1につき半周)
				GHOST:			0 0 0 0 0 0     #すり抜け設定(上/下/左/右/奥/手前)
			HITTEST_PARTS_END
			
		HITTEST_END
		
	HITTESTSET_END
	
	MOTIONSET	#モーション設定情報
		
		MOTION	#00==========待機==========(※MOTION~MOTION_ENDまでの括りを増やすことでモーションを追加出来ます。)
			LOOP_TIME:	40  #ループまでにかかるフレーム
			LOOP:		1   #ループするか否か(0:しない1:する)
			
			PARTS	#00部品1
				
				POSITION_ANIM #位置アニメーション
					TIME 0	0.0 0.0 2.0 10  #(開始フレーム/X/Y/Z/移動にかかるフレーム)
					TIME 10	2.0 4.0 0.0 20
				POSITION_ANIM_END
				
				ROTATION_ANIM #向きアニメーション
					TIME 0	0.0 0.5 0.0 30  #(開始フレーム/X/Y/Z/回転にかかるフレーム)
					TIME 30	0.0 1.0 0.0 60
				ROTATION_ANIM_END
				
			PARTS_END
			
		MOTION_END
		
	MOTIONSET_END
	
TYPE_END

END
